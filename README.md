# Git & Github Learning and Auditing Repository

This repository presents our group’s learning experience while auditing Google’s Introduction to Git and GitHub course on Coursera.

It is intended to demonstrate our practical understanding of version control concepts, Git and GitHub usage, and collaborative workflows through structured documentation, commits, and hands-on exercises aligned with each course module.
